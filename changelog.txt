Version 3.3.0
=============
* 버전 체계를 변경. (주 버전).(부버전).(패치)
* 다보리 멤버스도입
* 어드민 툴바에 다보리 메뉴 추가
* 아임포트 활성화시 PHP 에러 해결
* 아임포트 결제모듈: 다보리 세팅에서 설정한 "결제방식 지정"에 따라 "우커머스 > 설정 > 체크아웃" 설정을 변경.
  (결제방식 지정에 체크한 항목만 "우커머스 > 설정 > 체크아웃"에서 보임)
* IP Block 의 안정성 향상 (geolocate 데이터베이스 누락 시 자동 업데이트 로직 수정)

Version 3.2.5-alpha2
=====================
* 체크아웃 화면의 우편번호 검색 버튼이 작동하지 않는 버그 수정
* 주문 화면에서 성, 주소, 주/군의 validation check error 수정.

Version 3.2.5-alpha1
=====================
* 테스팅 도구 소스 트리로 포함.
* 포스트 내보내기 기능 포함.
* '직접 은행 이체 (BACS)' 결제 방법에서 입금인을 다르게 지정 가능.
* 체크아웃 화면의 '한국형 주소 찾기' 기능을 별도 모듈화.

Version 3.2.4-alpha3
