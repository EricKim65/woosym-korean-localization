== 설명 ==

"우커머스를 Cafe24보다 더 쉽게" 라는 모토로 만들어가고 있는 우커머스-심포니 한국형 통합 플러그인입니다. 

1. PG 결제 기능.

* 국내의 메이저급 PC 결제 대행사를 통합 지원합니다.
페이앱, LGU+, 카카오페이, KCP, 이니시스, 나이스정보. JT-Net, 올더게이트
(1) 쇼핑몰 자체 PG 결제 플러그인 기능 : 페이앱, KCP, 이니시스, 올더게이트
(2) “아임포트 서버” 연동 : LGU+, 카카오페이, KCP, 이니시스, 나이스정보. JT-Net
“아임포트 서버”를 이용하여 간접적으로 PG 결제하며 실제 결제 기능은 “아임포트 서버”에서 모두 처리하고 결과만 온라인 쇼핑몰로 넘겨줍니다.
(3) 신용카드, 계좌이체, 가상계좌, 모바일소액결제를 모두 지원합니다.
(4) 카카오 페이와 기존이 PG 결제 지원방식의 병행 지원이 가능합니다.
(5) 테스트 모드를 통해, 실제 결제 없이 결제여부를 확인할 수 있습니다.

2. 핵심기능

(1) 결제 페이지에서 다음(Daum) 우편번호 검색을 통한 주소입력을 제공합니다.
입력된 정보는 우커머스 배송주소에 저장되어, 재구매 시에 재입력할 필요가 없습니다.
(2)한국형 주소 찾기 :	한국형 주소 찾기와 결제페이지를 활성화합니다.
(3)회사명 입력 가능 :	사업자 대상 위주 판매의 경우 회사명을 입력 가능 설정을 합니다.
(4)한국 원화 표시 설정 :	우커머스->설정->일반->통화에서 대한민국(원)표시가 나오도록 합니다.
(5)SKU(상품코드) 사용해제	: SKU(상품코드) 사용을 해제합니다.
(6)상점으로 돌아가기 버튼해제 :	상점으로 돌아가기 버튼클릭 시 메인 홈으로 가게 합니다.
(7)관련상품표시 갯수, 관련상품표시 칸 수, 관련상품 필터 우선순위

3. 편의기능

(1) 배송추적 기능

* 우커머스 고객 주문 정보 확인시 관리자가 우커머스 관리자 모드에서 입력한 배송정로를 국내 택배회사 홈페이지와 연동하여 상품 배송중의 현재 상태를 직접 웹페이지에서 확인할 수 있도록 합니다.
* 지원 배송 업체 : 건영택배, 경동택배, 대신택배, 로젠택배, 우체국택배, 일양로지스,천일택배,한진택배,합동택배, 현대택배, CJ대한통운, KG로지스, KGB택배

(2)바로 구매 버튼 기능

우커머스에 없는 바로 구매 버튼 기능을 구현하여 국내 고객이 더욱 빨리 결제할 수 있도록 합니다.

4. 소셜 기능

페이스북과 네이버 아이디로 회원가입을 가능하게 합니다.

5. 차단보안기능

특별한 관리없이 악성댓글이나 악성트래픽이 대폭 감소합니다.
한국인 대상 사이트의 경우 한국,미국만 오픈해도 됩니다.

5. 마케팅 자동화 기능(개발중)

독립 웹사이트에서 마케팅 자동화 서버로 관련 데이타가 연동됩니다.
연동된 데이터가 빅데이터로 축적되어 마케팅 정보로 가공되어 매출 상승을 위한 기초데이터로 사용됩니다.
