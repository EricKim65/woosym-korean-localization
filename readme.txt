== 설명 ==

"우커머스를 Cafe24보다 더 쉽게" 라는 모토로 만들어가고 있는 우커머스-심포니 한국형 통합 플러그인입니다. 
추가되어 있는 기능는
(1) KCP 결제 모듈, 올더게이트, 이니시스 결제 모듈
(2) 올더게이트 결제 모듈
(3) 다음 주소 찾기 체크아웃
(4) 배송비와 배송 조회
(5) 페이스북, 네이버 계정 로그인
(6)기타 편의
(7)차단보안 기능

== 설치 ==

1. 웹사이트 또는 구매 사이트에서 플러그인을 다운 로드 하세요.
2. 워드프레스 대쉬보드에서 zip화일을  플러그인 > 추가 > 업로드해주세요.
3. 워드프레스 플러그인 메뉴에서 해당 플러그인을 활성화하여주십시요.

* 해당 플러그인 심포니 MVC  Plugin 을 전제로 하여 실행이 되므로 반드시 Symphony MVC Plugin을 먼저
  설치하고 활성화한 후 해당 플러그인을 추가로 설치해주시기 바랍니다.
